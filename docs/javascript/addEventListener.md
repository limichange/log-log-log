# addEventListener

```ts
button.addEventListener(
  'click',
  () => {
    // Perform an action
  },
  {
    // Invoke the callback at most once
    once: true,
    // Indicates the function will not call preventDefault useful for improving scroll performance
    passive: false,

    capture: true,
  }
)
```
