# V8 引擎的内存管理

运行 js 代码的时候从内容空间上分为堆空间和栈空间，这里栈空间就是创建执行上下文的调用栈。

```js
function foo() {
    let a = 1;
    let b = 'test';
    let c = {
        num: 3;
    }
    let d = c
}
foo()
```

当执行这段函数时，先编译，创建执行上下文，然后逐行执行赋值代码，基本数据类型的数据就存放在在了调用栈中
