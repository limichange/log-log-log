# CORS

浏览器对跨域请求区分为“简单请求”与“非简单请求”

（1) 请求方法是以下三种方法之一：

- HEAD
- GET
- POST

（2）HTTP 的头信息不超出以下几种字段：

- Accept
- Accept-Language
- Content-Language
- Last-Event-ID
- Content-Type：
- application/x-www-form-urlencoded、 multipart/form-data、text/plain
